Built-in template tags and filters

This document contains list of built-in template tags and filters.
The Grantlee libarary is used as an internal string template engine.
The syntax of Grantlee templates is the same as Django template system.
The Grantlee does not support all tags and filters of Django.
Therefore all actual tags and filters are provided below.
